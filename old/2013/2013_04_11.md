# [2013-04-11] PostgreSQL インストールしたらやること
_技術_

起動
```
# /etc/init.d/postgresql start
```
アカウント
```
# su – postgres
```
DB一覧
```
$ psql -l
```
DB接続
```
$ psql postgres
```
