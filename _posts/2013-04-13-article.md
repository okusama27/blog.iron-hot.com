普通のダンプ
```
$ pg_dump {dbname} > {filename}
```

アーカイブのダンプ
```
$ pg_dump -Fc {dbname} {filename}
```
tarのダンプ
```
$ pg_dump -Fｔ {dbname} {filename}
```
gzでダンプ
```
$ pg_dump  {dbname}  gzip >  {filename} gz
```
普通のリストア
```
$ pg_restore -C -d {dbname} {filename}
```
gzリストア
```
gunzip -c   {filename} gz  | psql -d {dbname} -U {username}
```
