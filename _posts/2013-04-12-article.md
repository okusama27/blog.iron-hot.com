ユーザ作成
```
$ createuser -P { username}
```
ユーザ削除
```
$ dropuser { username}
```
データベース作成
```
$ createdb {dbname}
```
データベース削除
```
$ dropdb  {dbname} 
```
起動
```
$ psql
$ psql  {dbname} 
$ psql  {dbname}  -U   { username}   -W
**メタコマンド**
```

テーブルの構造

終了
```
\q
```
