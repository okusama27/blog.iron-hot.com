```
rails new railstest –edge
cd railstest
bundle install
bundle exec rails server
```

下のメッセージが出たら成功

```
 INFO  WEBrick 1.3.1
 INFO  ruby 2.0.0 (2013-05-14) [x86_64-linux]
 INFO  WEBrick::HTTPServer#start: pid=3391 port=3000
```

`http://localhost:3000`

に接続
